# UNP_PROJECT_BI_WITH_POWER_BI_AIRBNB_ANALYSIS

I did this project in collaboration with UNP, where the primary objective was to analyze Airbnb data and derive meaningful business insights using Microsoft Power BI.
This project focuses on understanding pricing patterns, occupancy trends, and customer preferences across different locations. The dataset includes key attributes such as property type, price, availability, reviews, and location-based data. Using Power BI, I transformed raw data into interactive and visually engaging dashboards to support data-driven decision-making.
The analysis highlights important KPIs such as average price, booking trends, popular listings, and seasonal demand patterns. I created dynamic visualizations like bar charts, line graphs, and slicers to enable easy filtering and comparison across different regions and property types.
Key insights from this project include identifying high-demand areas, optimal pricing strategies, and customer behavior trends. Additionally, it helps in evaluating host performance and customer satisfaction levels.
Overall, this project demonstrates how Power BI can convert raw Airbnb data into actionable insights, helping hosts and businesses improve pricing strategies and maximize occupancy.”
